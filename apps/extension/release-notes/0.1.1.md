# Extension v0.1.1

## 변경 사항

- Popup header에 현재 extension version을 표시해 설치된 버전을 바로 확인할 수 있게 했습니다.
- Popup 준비 상태 문구를 Programmers 배지 동기화 흐름에 맞게 더 명확하게 다듬었습니다.

## 배포 메모

- Chrome Web Store 자동 게시 workflow를 통해 ZIP package upload와 publish 요청 제출을 진행합니다.
- Chrome Web Store 반영은 workflow 성공 직후가 아니라 Store review 승인 후 완료됩니다.
